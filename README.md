# Preparaci-n-de-Informaci-n-para-la-construcci-n-de-un-Modelo-de-Compra
El Notebook presenta la limpieza,preparación y validación  de variables para el input a un modelo, es la ingenieria, la selleción de variables y tratamiento 

Predicción de a compra o no compra de articulos

Este proyecto forma parte de mi formación en el Diplomado de Ciencia de Datos (UNAM). Consiste en un análisis end-to-end para la construcción de una TAD para un modelo predictivo, aplicando técnicas avanzadas de preprocesamiento, selección de variables y evaluación de métricas de desempeño.

Objetivo del Proyecto Limpiar y preparar una tabla analitica de datos para un modelo de clasificación capaz de predecir riesgos de salud basándose en indicadores biométricos, priorizando la precisión y la interpretabilidad de las variables.

Stack Tecnológico Lenguaje: Python

Manipulación de Datos: Pandas, NumPy

Visualización: Seaborn, Matplotlib

Estadística: Scipy (Pruebas KS)

Machine Learning: Scikit-learn (Imputación, Selección de Variables, Métricas)

Técnicas Especializadas: VarClusHi (Análisis de agrupamiento de variables)

Flujo de Trabajo (Pipeline)

Preprocesamiento e Imputación Limpieza de datos y manejo de valores nulos utilizando SimpleImputer.
Tratamiento de outliers y normalización de variables.

Ingeniería de Variables (Feature Engineering) Análisis de Varianza: Filtrado inicial mediante VarianceThreshold.
Reducción de Redundancia: Implementación de VarClusHi para identificar y eliminar variables altamente correlacionadas, optimizando la eficiencia del modelo.

Resultados La tabla es una limpieza efectiva en cuanto a variables y contenido, dejando las mejores para su integración al modelo de clasificación.
